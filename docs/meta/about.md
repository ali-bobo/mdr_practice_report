# 關於此作品集

## 作者背景

本作品集由一位具備 MDR（Managed Detection and Response）服務實務經驗的資安分析師維護，專注於：

- 威脅狩獵（Threat Hunting）
- 事件回應（Incident Response）
- 攻擊技術分析（MITRE ATT&CK 框架）
- 釣魚攻擊與社交工程分析

## 報告標準

每份報告在發布前通過以下審查：

1. **五層安全審查** — 直接識別 → 間接識別 → 文件元數據 → 自動掃描 → 人工終審
2. **10 項品質評估** — 技術準確性、ATT&CK 映射、時間軸完整性等
3. **匿名化確認** — 確保無法從報告內容還原客戶身份
